# Kunal Katre - Portfolio Website

A modern, interactive portfolio showcasing the work of Kunal Katre, a Lead Product Designer with 8.5 years of experience building scalable products in fast-paced environments.

## About This Project

This is a personal portfolio website built with React and TypeScript, designed to showcase professional experience, skills, and case studies. The site features:

- **Interactive Navigation**: Smooth transitions between sections and detailed case studies
- **Modern Design**: Clean, responsive layout with dark/light theme support
- **Project Showcase**: Detailed work portfolio with Co-WIN, Design Systems, Web Infrastructure, and Health Rate Monitor projects
- **Professional Sections**: Experience, education, awards, skills, and contact information

## Technologies Used

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: Vite build system

## View the Live Portfolio

See the portfolio in action: https://kunal-katre-portfolio.vercel.app/

## Run Locally

**Prerequisites:** Node.js (v16 or higher)

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up environment:**
   - Create a `.env.local` file in the root directory (optional configuration)

3. **Start the development server:**
   ```bash
   npm run dev
   ```

The portfolio will be available at `http://localhost:5173`

## Build for Production

To create a production-ready build:

```bash
npm run build
```

The optimized files will be in the `dist` directory.
